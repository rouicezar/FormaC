# CoreKnowledge 系统组件规范 & 异常状态库

## 1. 全局状态提示
- **加载中 (Loading)**: 骨架屏布局，米白色背景 (#F6F5F1) 配合淡灰渐变动画，确保视觉不突兀。
- **操作成功 (Success)**: 顶部居中提示条，墨绿色文字 (#2D4A43) 配以绿色对勾图标，3秒后自动消失。
- **操作失败 (Error)**: 顶部居中提示条，暗红色文字 (#B3261E) 配以感叹号图标，需手动关闭。
- **表单校验失败**: 输入框边框变为红色，下方显示红色辅助说明文字。

## 2. 核心异常状态
- **网络异常 (Network Error)**: 居中插画提示“网络连接中断，请检查设置”，提供“重新连接”按钮。
- **模型不可用 (Model Unavailable)**: 在问答流中显示琥珀色警告卡片：“模型服务连接超时，请联系管理员或稍后重试”。
- **404 页面不存在**: 居中显示“页面丢失了”，提供“返回首页”按钮。
- **403 无权限**: 居中显示锁图标及提示“暂无访问权限”，提供“返回首页”；只有存在真实申请流程时才显示“申请权限”。

## 3. 业务专项状态
- **扫描部分失败**: 在扫描详情列表中，异常文件以暗红色标识，并列出具体的解析/索引错误原因。
- **危险操作二次确认**: 弹窗背景弱化，标题为琥珀色警告，必须填写“确认原因”后才可激活执行按钮。
- **空状态**: 列表或搜索结果为空时，居中显示轻量级图标，并提供与当前权限和业务一致的引导。文档中心不得出现“新建”或上传入口。

## 4. 视觉规范细节
- **键盘焦点**: 所有可交互元素在聚焦时显示 2px 墨绿色描边。
- **按钮禁用**: 灰度处理，透明度降至 40%，鼠标指针显示为禁用。
