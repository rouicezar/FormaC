## 当前决策

- 以 `multimodal_agentic_rag` 的前后端分层为基础进行独立重组。
- 知识原文留在本地授权目录，应用不提供上传。
- 未进入白名单的飞书用户一律按外部客户处理。
- 敏感内容发送给云端模型的开关默认关闭。
- 所有用户端和管理端 Web 界面必须完全使用简体中文；模型正式名称、文件名和原文除外。
